Input Devices

Keyboards, mice, scanners — anything that turns an outside action into data the CPU can read. Every instruction cycle starts here or with data already stored in memory.

Control Unit

Fetches the next instruction, decodes what it means, and directs every other part of the computer on what to do next. Nothing executes without its say-so.

ALU

The Arithmetic Logic Unit does the actual work: addition, subtraction, comparisons, and logical operations, on whatever values the Control Unit hands it.

Registers

A handful of storage slots built directly into the CPU, holding the values currently being worked on — the smallest, fastest storage in the whole machine.

Cache Memory

A small, fast copy of data the CPU has used recently, sitting right beside it — so the next request for the same data doesn't need to go any further.

Primary Memory

RAM holds whatever program is currently running. Bigger than cache, still fast, but it empties out the moment the power goes off.

Secondary Memory

The SSD or hard disk stores everything permanently. It's enormous compared to RAM, but reaching it takes thousands of times longer.

Output Devices

Monitors, speakers, printers — where a finished result becomes something a person can actually see, hear, or hold.
